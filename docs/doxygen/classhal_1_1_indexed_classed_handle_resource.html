Detailed Description

template<typename THandle, typename TStruct, int16_t size, HAL_HandleEnum enumValue>
class hal::IndexedClassedHandleResource< THandle, TStruct, size, enumValue >

The IndexedClassedHandleResource class is a way to track handles. This version allows a limited number of handles that are allocated by index. Because they are allocated by index, each individual index holds its own mutex, which reduces contention heavily.]

Detailed Description

template<typename T>
class frc::VisionRunner< T >

A vision runner is a convenient wrapper object to make it easy to run vision pipelines from robot code. The easiest way to use this is to run it in a std::thread and use the listener to take snapshots of the pipeline's outputs.

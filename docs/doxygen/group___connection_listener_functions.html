◆ CreateConnectionListenerPoller()

NT_ConnectionListenerPoller nt::CreateConnectionListenerPoller ( NT_Inst  inst)

Create a connection listener poller. A poller provides a single queue of poll events. Events linked to this poller (using AddPolledConnectionListener()) will be stored in the queue and must be collected by calling PollConnectionListener(). The returned handle must be destroyed with DestroyConnectionListenerPoller().

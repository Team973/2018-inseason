◆ NT_PollRpc()

struct NT_RpcAnswer* NT_PollRpc ( NT_RpcCallPoller  poller,
size_t *  len 
)

Get the next incoming RPC call. This blocks until the next incoming RPC call is received. This is intended to be used with NT_CreatePolledRpc(void); RPC calls created using NT_CreateRpc() will not be serviced through this function. Upon successful return, NT_PostRpcResponse() must be called to send the return value to the caller. The returned array must be freed using NT_DisposeRpcAnswerArray().

Parameters
pollerpoller handle
lenlength of returned array (output)
Returns
Array of RPC call information. Only returns NULL if an error occurred (e.g. the instance was invalid or is shutting down).

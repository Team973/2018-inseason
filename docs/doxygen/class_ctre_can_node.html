bool CtreCanNode::ChangeTxPeriod ( uint32_t  arbId,
uint32_t  periodMs 
)
protected

Change the transmit period of an already scheduled CAN frame. This keeps the frame payload contents the same without caller having to perform a read-modify-write.

Parameters
arbIdCAN Frame Arbitration ID. Set BIT31 for 11bit ids, otherwise we use 29bit ids.
periodMsPeriod to transmit CAN frame. Pass 0 for one-shot, which also disables that ArbID's preceding periodic transmit.
Returns
true if scheduled job was found and updated, false if there was no preceding job for the specified arbID.
